# Portrait Shoot Images

## How to Add Images

1. Add your portrait photography images to this folder
2. Name them sequentially: `image-1.jpg`, `image-2.jpg`, `image-3.jpg`, etc.
3. Optionally, create thumbnails: `thumb-1.jpg`, `thumb-2.jpg`, etc.
4. Supported formats: JPG, JPEG, PNG, WebP
5. Recommended size: 1920x1080 or similar aspect ratio
6. Images will automatically appear in the gallery

## Image Optimization Tips

- Compress images before uploading (use tools like TinyPNG or ImageOptim)
- Recommended file size: under 500KB per image
- Use WebP format for better compression
- Create thumbnails at 400x300 for faster loading
